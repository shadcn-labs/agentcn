**agentcn** is a registry of complete, production-ready agent recipes for the
[Eve](https://vercel.com/docs/eve) and [Flue](https://flueframework.com)
frameworks. It is **not** a UI component registry — each recipe distributes the
backend agent files (instructions, tools, skills, workflows) that you copy
straight into your Eve or Flue project with the shadcn CLI.

## How It Works

Browse the [Agents](/docs/agents) section, pick a recipe, and run its install
command:

```bash
npx shadcn@latest add https://agentcn.vercel.app/r/eve/deep-search
```

The CLI writes the recipe's files into your project at their target paths. From
there the code is yours — swap the model, extend the tools, or reshape the
output to fit your use case.

## Why Recipes Instead of Packages

The shadcn philosophy gives you **ownership** of the code. Instead of importing
an opaque agent from a package, you get the full source — agent definition,
instructions, tools, and skills — and can customize every part without fighting
library constraints.

## Frameworks

- **[Eve](https://vercel.com/docs/eve)** — a filesystem-first framework for
  durable backend AI agents that run on Vercel Functions.
- **[Flue](https://flueframework.com)** — an agent runtime with first-class
  subagents, skills, and bounded workflows.

Every recipe ships in both flavors where it makes sense, so you can adopt the
pattern in whichever framework you already use.

## What's Included

- 📦 **Copy-paste recipes** — install with `npx shadcn add`
- 🧪 **Live previews** — run each agent right from its docs page
- 📚 **Documentation site** — powered by Fumadocs
- 🤖 **Agent ready** — ships with `llms.txt`, `llms-full.txt`, API catalog
  routes, and agent skill discovery endpoints
- 🔊 **[Web audio feedback](https://audio.raphaelsalaja.com/)** — built-in
  interaction sounds powered by `@web-kits/audio`
- 📳 **[Web haptics](https://haptics.lochie.me/)** — optional haptic feedback via
  `web-haptics`
- ✨ **[Motion animations](https://motion.dev/)** — animated UI states and
  transitions
