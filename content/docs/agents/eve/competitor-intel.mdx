The **Competitor Intelligence Agent** takes a list of competitor URLs, scrapes
each page, and analyzes positioning, pricing, and key features. It returns a
single structured report you can drop into a dashboard, a sales brief, or a
weekly digest. Reach for it whenever you need a repeatable, grounded read on how
a set of competitors present themselves — without copy-pasting pages by hand.

<FrameworkDocTabs agent="competitor-intel" current="eve" />

The **Eve** recipe runs the loop inside a single durable session: it works
through each URL sequentially, **scrapes** the page to markdown with the
`scrape_url` tool, **analyzes** the content to extract positioning, pricing
tiers, and notable features — reporting only what is actually on the page — then
**synthesizes** the findings into a structured report with a top-level summary.

## Install

```bash
npx shadcn@latest add https://agentcn.sh/r/eve/competitor-intel
```

## Customization

- **Swap the model.** Edit `agent/agent.ts`. Any model the Eve gateway supports
  works — for example `anthropic/claude-sonnet-4-6` for deeper analysis or
  `anthropic/claude-haiku-4-5` for speed.
- **Extend the tools.** Add another file under `agent/tools/` to enrich the
  report — e.g. a `screenshot` tool, a pricing-page crawler, or a CRM lookup.
- **Add more URLs.** The recipe processes URLs sequentially, so you can pass as
  many competitors as you like — paste one URL per line in the preview above.
- **Reshape the report.** Adjust the output schema in `synthesize_report` to
  capture additional fields such as funding, headcount, or sentiment.
