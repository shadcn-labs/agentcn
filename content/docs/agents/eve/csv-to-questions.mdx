The **CSV to Questions Agent** takes a CSV dataset and turns it into a set of
sharp, answerable analytical questions. It first summarizes the data — columns,
types, ranges, patterns — to compress large files and avoid token-limit errors,
then generates questions a data analyst would actually ask. Reach for it to
kickstart exploratory analysis or build study material from raw data.

<DocsBaseSwitcher base="eve" agent="csv-to-questions" />

The **Eve** recipe runs in a single session: it **loads** the CSV with the
`fetch_csv` tool, **summarizes** it into a compact column profile, then
**generates** analytical questions grounded in the columns that actually exist.

## Install

```bash
npx shadcn@latest add https://agentcn.vercel.app/r/eve/csv-to-questions
```

## Prerequisites

- An [Eve](https://vercel.com/docs/eve) project (`npx eve@latest init my-agent`).
  The recipe installs files under `agent/`, which Eve discovers automatically.
- The `eve` package (installed automatically as a dependency of the recipe).

## Customization

- **Read local files.** Swap `fetch_csv` to read from disk or object storage
  instead of a URL.
- **Control the question style.** Edit `instructions.md` to bias toward trends,
  comparisons, or hypothesis tests.
- **Swap the model.** Edit `agent/agent.ts`. A large-context model handles wide
  datasets without truncation.
- **Chunk huge files.** Summarize in row batches and merge the profiles before
  generating questions.
