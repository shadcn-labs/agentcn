## Summary

The **OpenUI Assistant** is a conversational agent powered by OpenUI's
real-time UI generation. It can render interactive React components inline
during a conversation — charts, forms, data tables, and more — using the
OpenUI Lang component library. It also includes a timezone-aware time tool
as a baseline demonstration of the tool-call path.

## Composition

```text
agent/
├── agent.ts                  # Agent definition (model + OpenUI deps)
├── instructions.md           # Base identity instructions
├── channels/
│   └── eve.ts                # HTTP channel with anonymous auth
├── instructions/
│   └── openui.ts             # Dynamic OpenUI Lang instructions
└── tools/
    └── get_current_time.ts   # Timezone-aware current time tool
```

## Customization

- **Swap the model.** Edit `agent/agent.ts` to use a different provider or model.
- **Add authentication.** Replace `none()` in `channels/eve.ts` with `bearer()` or `basic()`.
- **Add more tools.** Extend the agent with additional capabilities.
- **Customize instructions.** Edit `instructions.md` or `instructions/openui.ts`.
