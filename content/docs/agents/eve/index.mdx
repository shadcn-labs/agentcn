[Eve](https://vercel.com/docs/eve) is a filesystem-first framework for durable
backend AI agents that run on Vercel Functions. These recipes install the agent
files (instructions, tools, and skills) straight into your Eve project's
`agent/` directory with the shadcn CLI.

## Available agents

- **[Competitor Intelligence](/docs/agents/eve/competitor-intel)** — scrapes and
  analyzes competitor pages inside a single durable session, then returns a
  structured intelligence report.

More Eve recipes coming soon. Each one includes an install command,
prerequisites, a live preview, and a breakdown of every file it installs.
