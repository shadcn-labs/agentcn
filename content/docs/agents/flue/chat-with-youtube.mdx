## Summary

The **Chat with YouTube Agent** answers questions about a video using its
transcript. It pulls the video's metadata and full transcript, then summarizes,
answers follow-ups, and generates chapter timestamps — each citation a clickable
link straight to that moment. Reach for it to skim long talks, lectures, or
podcasts without watching end to end.

## Customization

- **Reliable fetching.** Metadata and transcripts are pulled through the [Bright
  Data Web Unlocker](https://docs.brightdata.com/scraping-automation/web-unlocker),
  which handles YouTube at scale with no rate limits or blocks. Set
  `BRIGHTDATA_API_KEY` (and optionally `BRIGHTDATA_UNLOCKER_ZONE`).
- **Add chapters export.** Add a tool that writes the chapter list as a
  description or VTT file.
- **Handle missing transcripts.** Fall back to an STT service when captions are
  unavailable.
- **Add a workflow.** Wrap the tools in a workflow with a typed chapters schema.
- **Swap the model.** Change the `model` field on the agent.
