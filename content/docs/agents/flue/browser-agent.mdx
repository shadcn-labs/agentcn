## Summary

The **Browser Agent** completes tasks on the live web by driving a real Chromium
browser. It uses a snapshot-and-selector loop — the model sees a stable list of
interactive elements instead of brittle CSS — to navigate, click, and type. Reach
for it to automate flows, scrape interactive pages, or test UIs.

## Customization

- **Managed remote browser.** `lib/browser.ts` connects to the [Bright Data
  Browser API](https://docs.brightdata.com/scraping-automation/scraping-browser) —
  a managed, anti-bot-resistant Chromium with no local infra. Set
  `BRIGHTDATA_BROWSER_AUTH` to your zone `username:password`, or point
  `BROWSER_CDP_URL` at any other CDP endpoint. With neither set it falls back to
  local Chromium for development.
- **Richer snapshots.** Extend `lib/browser.ts` to include roles, ARIA, or
  bounding boxes.
- **Add actions.** Add tools for scrolling, file uploads, or screenshots.
- **Swap the model.** Change the `model` field on the agent.
