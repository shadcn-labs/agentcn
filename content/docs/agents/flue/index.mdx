[Flue](https://flueframework.com) is an agent runtime with first-class
subagents, skills, and bounded workflows. These recipes install the agent,
workflow, tool, and skill files straight into your Flue project with the shadcn
CLI.

## Available agents

- **[Competitor Intelligence](/docs/agents/flue/competitor-intel)** — scrapes and
  analyzes competitor pages with a fan-out subagent pattern, then returns a
  structured intelligence report.

More Flue recipes coming soon. Each one includes an install command,
prerequisites, a live preview, and a breakdown of every file it installs.
