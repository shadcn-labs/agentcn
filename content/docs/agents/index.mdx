Agent recipes are complete, copy-paste backend patterns for the
[Eve](https://vercel.com/docs/eve) and [Flue](https://flueframework.com)
frameworks. Each recipe ships its instructions, tools, skills, and (for Flue)
workflows, installable with the shadcn CLI.

## Available agents

- **[Competitor Intelligence](/docs/agents/competitor-intel)** — scrapes and
  analyzes competitor pages, then returns a structured intelligence report.
  Available for both Eve and Flue.

More recipes coming soon. Each one includes an install command, prerequisites, a
live preview, and a breakdown of every file it installs.
